# excel-pivot-analysis-2
# 📊 Pivot Table Financial Analysis Practice

## Overview

This project contains hands-on Excel Pivot Table exercises focused on financial reporting, data analysis, and business insights.

The objective of this project is to practice creating Pivot Tables, Pivot Charts, calculated metrics, and interactive reports using financial transaction data.

## Skills Demonstrated

* Pivot Table Creation
* Data Summarization
* Filtering and Sorting
* Grouping Dates by Month and Year
* Slicers and Interactive Reporting
* Financial Statement Analysis
* Year-over-Year (YoY) Revenue Analysis
* Margin Analysis
* Trial Balance Reporting
* Multi-Level Financial Reports

## Exercises Completed

1. Sales by Region
2. Transaction Count by Month
3. Category-wise Financial Summary
4. Product and Region Filtering
5. Trial Balance Summarization
6. Statement-wise Analysis (P&L vs Balance Sheet)
7. Year-over-Year Revenue Comparison
8. Department-wise Expense Analysis
9. Multi-Level P&L Statement
10. Month-End Balance Sheet Reporting
11. Retained Earnings Calculation
12. Margin Analysis by Product
13. Variance Analysis
14. Custom Sorting of Financial Statements
15. Multi-Dimensional Pivot Analysis

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Formulas
* GETPIVOTDATA Functions

## Key Learnings

* Creating dynamic reports using Pivot Tables
* Analyzing financial data from multiple perspectives
* Building interactive reports with slicers
* Understanding financial reporting concepts
* Generating business insights from raw data
